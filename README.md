# Essential JS 2 for React - Workflow Automation
A Workflow Automation Designer converts process definitions into interactive diagrams, helping users easily visualize tasks, flows, and dependencies for streamlined automation.

## Prerequisite 

Install Node version - 22.14.0 from https://nodejs.org/en/download/ 

## To Install all packages

Use the below command to install all dependent packages.

```
npm install
```

## To Run the application

Run `npm start` for a dev server. Navigate to `http://localhost:8080/`.
